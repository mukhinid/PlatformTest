Нужно реализовать на ASP.NET Core сервер для хранения файлов, который поддерживает несколько типов хранилищ. АПИ сервера имеет три метода:
* добавление файла
* удаление файла
* получение файла

Каждый метод принимает идентификатор хранилища в качестве параметра.
Примеры хранилищ:
* "локальное" хранит все файлы в одной директории
* "ftp" хранит все файлы на FTP сервере
